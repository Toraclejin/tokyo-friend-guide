# 도쿄 친구 가이드

우에노 기준 도쿄 자유여행 가이드 — 구역별 **관광지·맛집·카페** + **디즈니랜드 당일 원정**.

- 구역별 인터랙티브 지도(장소 핀) + 각 장소 구글맵 링크
- 카테고리 / ★추천 필터
- 디즈니랜드 가는 법(3가지) + 호텔→디즈니랜드 구글 길찾기 버튼
- 모바일 우선, 단일 HTML 파일

## 사용법

`index.html` 을 브라우저로 열면 됩니다. (지도 표시에는 인터넷 필요, 나머지는 오프라인에서도 동작)

장소를 추가/수정하려면 `index.html` 안의 `GUIDE_DATA` 객체 또는 `tokyo_guide_data.json` 을 고치면 됩니다.
